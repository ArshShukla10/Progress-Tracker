# Changelog

All notable changes to **NEXUS** will be documented in this file.

The format is inspired by *Keep a Changelog* and follows semantic versioning.

---

## [v0.6.0] - AI Learning Engine

### Added

* Provider-based AI architecture
* Gemini provider (default)
* Provider abstraction for OpenAI, Anthropic and Ollama
* AI Service layer
* Prompt Builder
* Context Builder
* Response Parser
* AI Actions
* AI Settings
* AI Topic Panel
* Ask AI panel
* AI Loading and Error components

### Improved

* Learning Workspace AI integration
* Topic-level AI cards
* Context-aware AI prompts
* Provider-independent architecture

---

## [v0.5.3] - Skills Architecture

### Added

* Skills section
* Skill Service
* DSA as first unlocked skill
* Locked skill placeholders
* Dashboard Skills section
* Sidebar Skills navigation
* Cross-link from Data Structures to DSA

### Changed

* Removed Computer Graphics from Semester III
* Removed DSA Theory
* Moved DSA from Academics to Skills
